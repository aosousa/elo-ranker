# ELO Ranker

React application to vote on different rankings and save their results. Ranking is calculcated using the Elo rating system.

### Tech Stack
#### Back end
* Node.js (+ Express.js, Sequelize)
* MySQL

#### Front end
* React (+ Redux)
* TypeScript
